# HowsWeatherApp

App that shows the weekly weather for a city

## Built With

- HTML
- CSS (SASS)
- Parcel.js

## Getting Started

```
npm install
```

Install npm depedencies.

```
npm run dev
```

Runs the server in development mode.

```
npm run build
```

Generates a folder named build with a project version ready for deployment.
